# Roadmap

## Fase 1: MVP vendible

- Backend modular Spring Boot.
- Frontend Angular para crear y aprobar borradores.
- Docker Compose completo.
- Multi-tenant básico.
- Eventos RabbitMQ base.

## Fase 2: RAG real

- Ingesta de documentos a MinIO.
- pgvector.
- Citaciones y permisos documentales.

## Fase 3: Conectores

- CRM, email, calendario y tickets.
- Tool Router con aprobaciones por riesgo.

## Fase 4: Enterprise

- OIDC completo.
- RLS PostgreSQL.
- Billing por tenant.
- Extracción selectiva de microservicios.
